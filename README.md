# SWJTU-HELIOS      **vision group**
## 1. 运行命令
|  命令 |说明   |
| ------------ | ------------ |
|   -blue| 将蓝色装甲板视为识别目标  |
|   -red|  将红色装甲板视为识别目标 |
|  -hsv | 打开hsv颜色模式  |
|   -help|输出运行命令说明  |
|   -box| 展示目标装甲板  |
|   -boxes|  展示所有可能的目标装甲板 |
|  -blobs |  展示目标灯柱 |
|   -origin|  展示原视频 |
|  -camera |使用摄像头   |
|   -save| 保存视频  |
|   -save2| 保存目标区域截图  |
|   -energy|展示能量机关   |
|   -debug  | 展示二值化图像、视频，装甲板、能量机关|
其中，颜色项必选！！！
